# ClassroomInput

Classroom marks and notes recording. Define custom form types, manage classrooms
and students, and record structured observations.

## Features

- Define custom form types with configurable columns
- Manage classrooms and pupil lists
- Record marks, notes, and observations per session
- Spreadsheet-style data entry
