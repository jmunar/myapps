# MindFlow

Thought capture and mind mapping. Quickly jot down ideas, organise them into
categories, visualise connections on a D3-powered map, and turn thoughts into
actions.

## Features

- Quick thought capture via inbox
- Category tree with colors and icons
- D3.js-powered mind map visualisation
- Nested thoughts (parent/child)
- Comments on thoughts
- Action items with due dates and priorities
